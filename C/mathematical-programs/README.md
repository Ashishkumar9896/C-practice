# Mathematical Programs

This directory contains C programs for various mathematical calculations:

- `factorial.c` - Factorial calculation
- `armstrong.c` - Armstrong number check
- `powerof2.c` - Power of 2 calculations
- `sum_of_first_n_integer.c` - Sum of first n integers
- `sumofdigits.c` - Sum of digits calculation
- `sumofeven.c` - Sum of even numbers
- `sumofsquareofno.c` - Sum of squares of numbers
- `leapyear.c` - Leap year calculation
